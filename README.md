# mitt-forste-repo

Et lite testprosjekt for å lære GitHub-flyten.

## Hva er dette?

Dette er et enkelt prosjekt laget for å øve på pull requests med Claude Code.

## Kom i gang

Klon repoet og utforsk filene.

## Bidra

1. Lag en ny branch: `git checkout -b min-endring`
2. Gjør endringene dine
3. Commit: `git commit -m "Beskriv hva du endret"`
4. Push og åpne en pull request
